**Daily Reddit Game - Spot the AI-Generated Comment**

Can you tell which comment was written by AI? Test your detection skills daily!

## How to Play

1. Read 5 comments responding to a Reddit prompt
2. 4 are real Reddit comments, 1 is AI-generated
3. Pick the one you think is fake
4. Get instant feedback and see how you compare to others

## Features

- **Daily Puzzles** - New puzzle every day at midnight UTC
- **Streak Tracking** - Build your consecutive correct answer streak
- **Difficulty Progression** - Monday (easy) ‚Üí Friday (hard) ‚Üí Weekend (expert)
- **Statistics** - See how your guesses compare to other players
- **Share Results** - Share your streak without spoiling the answer
